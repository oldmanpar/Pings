# 🌟 Pings - シンプルなPing監視ツール

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 💡 概要

Pingsは、複数のネットワークホストの疎通性をリアルタイムで監視し、詳細な統計情報を記録するために開発された、**軽量かつ高機能なWindowsフォームアプリケーション**です。

ネットワークの基本的なヘルスチェックや、障害発生時の状況把握に役立ちます。

---

## 💻 主な機能

* **マルチホスト監視:** 複数のIPアドレスやドメインを同時に監視します。
* **リアルタイムステータス表示:** OK / Down / 復旧の状態を色で視覚的に識別できます。
* **詳細な統計:** 応答時間の平均/最小/最大、失敗回数、連続失敗時間などを記録します。
* **障害イベントログ:** Down開始時刻と復旧時刻、ダウン時間を正確に記録し、エクスポート可能です。
* **柔軟なファイル入出力:**
    * 監視対象リストの読み込みは、プログラム保存形式（CSV）に加え、**ExPing形式（スペース/タブ区切り）**のテキストファイルに対応しています。
    * 監視結果、障害ログのCSVファイル保存に対応しています。

## 🛠️ 利用方法 (実行ファイルを使用する場合)

1.  **リリースのダウンロード:**
    リポジトリの **[Releases]** ページから、最新バージョンの **`Pings_vX.X_Release.zip`** ファイルをダウンロードします。
2.  **展開:**
    ダウンロードしたZIPファイルを任意の場所に展開（解凍）します。
3.  **実行:**
    展開したフォルダ内の **`Pings.exe`** をダブルクリックして起動します。
